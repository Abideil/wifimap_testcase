Buttons for calling data aggregation

Button 1 - Count wifi hotspots created by user
Button 2 - Count wifi hotspots with geoposition created by user
Button 3 - Count the total number of created by user hotspots during various time periods
Button 4 - Count the total number of created by user hotspots with desired score values
Button 5 - Count how many hotspots the user has to which there were more than 1, 5 and 10 unique connections during various time periods
